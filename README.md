# Image and Links

## Description

This is a beginner HTML project that demonstrates how to create clickable images using hyperlinks.

## Features

* Display images using the `<img>` tag
* Create hyperlinks using the `<a>` tag
* Open links in a new tab

## Technologies Used

* HTML5

## What I Learned

* How to add images to a webpage
* How to create links
* How to make an image clickable
